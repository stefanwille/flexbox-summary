Flexbox Summary

Main Axis Direction - flex-direction

Put Flex Items in a Row - the Default

flex-direction: row

Put Flex Items in a Column

flex-direction: column

Alignment on the Main Axis: justify-content

Put Flex Items at the Beginning

justify-content: flex-start

Put Flex Items at the End

justify-content: flex-end

Center Flex Items

justify-content: center

Spread Flex Items evenly

justify-content: space-around

Spread Flex Items evenly

justify-content: space-between

Alignment on the Cross Axis: align-items

The same options from "justify-content" exist for the cross axis via "align-items".

Not all options are demonstrated here, just two.

Put Flex Items at the Beginning of the Cross Axis

align-items: flex-start

Put Flex Items at the End of the Cross Axis

align-items: flex-end

Put Flex Items at the Center of the Cross Axis

align-items: center

And so on...
